# 🎨 Image Generation with Stable Diffusion

## 📌 Overview
This project demonstrates AI-powered image generation using 
pre-trained Stable Diffusion model via HuggingFace Diffusers library.

## 🛠️ Technologies Used
- Python
- HuggingFace Diffusers
- Stable Diffusion v1.4
- Google Colab (Free GPU)
- PyTorch

## 🖼️ Generated Images
| Prompt | Output |
|--------|--------|
| Himalayan Sunset | landscape.png |
| Fantasy Dragon | dragon.png |
| Underwater City | underwater_city.png |
| Majestic Wolf | wolf.png |
| Space Fantasy | space.png |
| Magical Forest | forest.png |

## 🚀 How to Run
1. Open the notebook in Google Colab
2. Enable GPU (Runtime → Change runtime type → T4 GPU)
3. Run all cells in order
4. Change prompts to generate your own images!
